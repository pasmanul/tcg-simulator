# dmapp 操作説明書

デュエルマスターズ非公式カードゲームシミュレーター。

---

## 起動・初期設定

### ウィンドウ構成

アプリを起動すると 2 つのウィンドウが開く。

| ウィンドウ | タイトル | 用途 |
|---|---|---|
| フィールドウィンドウ | デュエルマスターズ — フィールド（画面共有用） | 対戦相手や配信に見せる公開画面 |
| 手札ウィンドウ | 手札・デッキ（非公開） | 自分だけが見る非公開画面 |

### デッキの読み込み

1. **手札ウィンドウ** 上部の「デッキ読み込み」ボタンをクリック
2. `data/decks/` フォルダ内のデッキ JSON を選択
3. デッキ名が上部ラベルに表示されれば読み込み完了

前回読み込んだデッキは次回起動時に自動で復元される。  
デッキが 40 枚であれば、起動時に自動でゲーム開始状態（シールド 5 枚・手札 5 枚）が配られる。

---

## ゲームの開始

1. デッキを読み込む
2. フィールドウィンドウの「初期状態にリセット」ボタン（またはメニュー「ファイル → 初期状態にリセット」）をクリック
3. 確認ダイアログで「Yes」→ デッキがシャッフルされ、シールド 5 枚・手札 5 枚が配られる

---

## ゾーン一覧

| ゾーン | 表示場所 | 公開状態 |
|---|---|---|
| バトルゾーン | フィールド上段 | 公開 |
| シールドゾーン | フィールド中段左 | 非公開（裏向き） |
| 山札 | フィールド中段中央 | 非公開（枚数のみ表示） |
| 墓地 | フィールド中段右 | 公開 |
| マナゾーン | フィールド下段左 | 公開 |
| 手札 | 手札ウィンドウ上段 | 非公開（自分のみ閲覧可） |
| 保留 | 手札ウィンドウ中段 | 非公開（一時置き場） |

## 基本操作

### カードの移動（ドラッグ＆ドロップ）

- カードをドラッグして別のゾーンにドロップすると移動できる
- **山札へのドロップ**：「山札の一番上」か「一番下」を選ぶダイアログが表示される

### タップ / アンタップ

- **バトルゾーン・マナゾーンのカード**をクリックするとタップ / アンタップが切り替わる
  - タップ = 90° 回転（黄色い枠で強調表示）
  - マナゾーンのカードは上下反転（180° 回転）で表示される

### 全タップ / 全解除

各ゾーン右上に表示されるボタンで一括操作できる。

| ボタン | 効果 |
|---|---|
| 全タップ | そのゾーンの全カードをタップ状態にする |
| 全解除 | そのゾーンの全カードをアンタップ状態にする |

### カードの並び替え（同一ゾーン内）

- ゾーン内でカードをドラッグ＆ドロップすると順序を変更できる
- キーボードの **← →** キー（カードにホバー中）でも左右に移動できる

### バトルゾーンの上段 / 下段

バトルゾーンは上段・下段の 2 行構成。

- ドロップ位置の上半分 → 上段、下半分 → 下段に配置される
- カードにホバー中に **↑ ↓** キーで行を変更できる
- 右クリックメニュー → 「上段へ移動」「下段へ移動」でも変更できる

---

## 右クリックメニュー（カード）

カードを右クリックすると以下のメニューが表示される。

| 項目 | 説明 |
|---|---|
| タップ / アンタップ | カードの向きを切り替える |
| 表向きにする / 裏向きにする | カードの表裏を切り替える |
| マーク | カードに色マーク（赤・青・黄・緑・紫）を付ける／外す |
| 公開する / 非公開にする | 手札ゾーンのカードをフィールドに公開する（手札ゾーンのみ） |
| 上段へ移動 / 下段へ移動 | バトルゾーン内で行を変更する（バトルゾーンのみ） |
| スタックを確認 | 進化スタックの内容を表示する（スタックがある場合のみ） |
| 一番下を切り離す | 進化スタックの最下カードを取り出す（スタックがある場合のみ） |
| 山札をサーチ... | サーチダイアログを開く（山札のみ） |

---

## キーボードショートカット（カードにホバー中）

カードにマウスを重ねた状態でキーを押すとゾーン移動できる。

| デフォルトキー | 操作 |
|---|---|
| `B` | バトルゾーンへ移動 |
| `M` | マナゾーンへ移動 |
| `G` | 墓地へ移動 |
| `H` | 手札へ移動 |
| `S` | シールドゾーンへ移動 |
| `D` | ドロー（山札から手札へ 1 枚引く） |
| `R` | ゲームをリセット（初期状態にリセット） |
| `←` `→` | 同ゾーン内でカードを左右に移動 |
| `↑` `↓` | バトルゾーンで上段 / 下段を切り替える |

※ キーバインドはカスタマイズ可能（後述）。

---

## 複数カードの選択とまとめ移動

1. **Shift + クリック** で複数カードを選択（青い枠でハイライト表示）
2. 選択中のいずれかのカードをドラッグ → 選択した全カードをまとめて移動

---

## 進化（スタック）

バトルゾーンで、**ドロップ先に既存のカードがある場合**、その上に重ねて進化扱いになる。

- 進化元のカードは下に積まれ、スタック枚数バッジ（赤い数字）が表示される
- 右クリック → 「スタックを確認」でスタック内容を確認できる
- 右クリック → 「一番下を切り離す」でスタックを分離できる
- バトルゾーン以外に移動すると、スタックは分離されて個別に追加される

---

## ホバーズーム

表向きのカードにマウスを 0.6 秒静止させると、カード画像を拡大表示するウィンドウが開く。

---

## ゾーンの展開表示

カードが重なっているゾーン（重なり時はゾーン下部に「ダブルクリックで展開」と表示）で、**カード以外の部分をダブルクリック**すると、カードを並べた展開ダイアログが開く。

---

## ドロー

以下のいずれかの方法でドローできる。

- フィールドウィンドウ：山札パネルの「ドロー」ボタン
- 手札ウィンドウ：「ドロー」ボタン
- キーボードショートカット `D`（デフォルト）

---

## 山札のシャッフル

フィールドウィンドウの山札パネル「シャッフル」ボタンをクリック。  
シャッフル時に白フラッシュアニメーションが表示される。

---

## 山札サーチ

山札のカードを右クリック → 「山札をサーチ...」でサーチダイアログを開く。

1. カード名・文明・タイプで絞り込みができる
2. 目的のカードをクリックして選択（複数選択可）
3. 移動先（手札 / 保留 / マナゾーン / 墓地）を選んで「確定」

---

## 手札のソート

手札ウィンドウ上部「ソート」ボタンをクリックするとマナコスト順に整列する。

## バトルゾーンのソート

バトルゾーン右上「ソート」ボタンでバトルゾーンのカードをソートする。

---

## ダイス

フィールドウィンドウ左上「ダイス」ボタンをクリックするとダイスダイアログが開く。

- **d4 / d6 / d8 / d10 / d12 / d20** の 6 種類から選択
- 「振る！」ボタンをクリック（またはウィンドウにフォーカスした状態で **Enter** / **Space**）でロール
- 結果はアクションログにも記録される
- 直近 10 件の履歴が表示される

---

## セーブ / ロード

| 操作 | 方法 |
|---|---|
| 試合を保存 | メニュー「ファイル → 試合を保存」（または `Ctrl+S`） |
| 試合をロード | メニュー「ファイル → 試合をロード…」 |

保存ファイルは `data/saves/` に JSON 形式で保存される。

---

## アンドゥ

直前の操作を最大 50 回までアンドゥできる。

- メニュー「ファイル → アンドゥ」
- ショートカット `Ctrl+Z`

---

## デッキ管理

メニュー「ファイル → デッキ管理を開く」またはフィールド・手札ウィンドウの「デッキ管理」ボタンでデッキエディタを開く。

- カードライブラリからカードをドラッグしてデッキに追加
- デッキ内のカードをドラッグして削除
- デッキの保存・新規作成・削除が可能
- カードのソート（マナ↑ / マナ↓ / 名前順 / タイプ順）が可能

---

## キーバインドの変更

メニュー「設定 → キーバインド設定…」でショートカットキーを変更できる。

変更可能なアクション：

| アクション | デフォルトキー |
|---|---|
| バトルゾーンへ移動 | `B` |
| マナゾーンへ移動 | `M` |
| 墓地へ移動 | `G` |
| 手札へ移動 | `H` |
| シールドへ移動 | `S` |
| ドロー | `D` |
| ゲームリセット | `R` |

設定は `data/config.json` に保存され、次回起動時も維持される。

---

## アクションログ

フィールドウィンドウ右側のパネルに操作履歴が記録される。  
非公開ゾーン（手札・シールド・山札・保留）に関わる操作はカード名が「?」と表示される。
